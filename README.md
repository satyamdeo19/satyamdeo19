## How to run the projects (general)

> The three featured projects follow similar setup patterns. Check each repository's README for repo-specific instructions.

1. Clone the repo: `git clone <repo-url>`
2. Install server dependencies: `cd server && npm install` (or `pip install -r requirements.txt` for Flask)
3. Install client dependencies: `cd client && npm install`
4. Setup environment variables (DB URI, JWT secret, Cloudinary keys, Mapbox token)
5. Start server: `npm run dev` (or `python app.py`)
6. Start client: `npm start`
